Website Project Kanban View

Geminate comes with a feature to introduce kanbanview for project and their related task on website. it will show the progress of each task based on their stage and due date.







Available Key Features

  • Project kanbanview on website
  • Project task kanbanview on website
  • Monitor progress of task based on their stage direct from website
  • Easy access for portal user for analytic purpose
  • Graphical view and more user friendly
